# Simple example for Sanic [REST Framework](https://github.com/Tioit-Wang/sanic-rest-framework)  

## Run example
```shell
git clone https://github.com/Tioit-Wang/srf_simple_example.git 

cd srf_simple_example

pip install pipenv

pipenv install

pipenv run python run.py
```
